## Using the Charts

### Viewing Charts
1. Go to **Reports** tab
2. Scroll down to see all charts
3. Charts auto-update when you add loans/payments/savings

### Filtering Data

#### Date Range Filter
```
1. Find the chart you want to filter
2. Click the date input fields in the header
3. Select start and end dates
4. Chart updates automatically
```

**Available on:**
- Savings Growth Trend
- Top Members by Savings

#### Period Comparison
```
1. Go to "Period Comparison Analysis" chart
2. Select from dropdown:
   - Month-over-Month (compare weeks)
   - Year-over-Year (compare months)
3. Chart updates instantly
```

## Exporting Data

### Export as Image (PNG)
```
1. Find the chart you want to export
2. Click [Export] button in the header
3. File downloads automatically
4. Use for presentations, emails, etc.
```

**Available on all charts**

### Export as PDF Report
```
1. Scroll to bottom of Reports section
2. Click [Export Complete PDF Report]
3. Wait 2-3 seconds for generation
4. File downloads as SACCO_Analytics_Report.pdf
```

**Includes:**
- All 5 charts in one document
- Generation date
- Professional formatting
- High-quality images

## Understanding the Charts

### Savings Growth Trend
- **Blue line** = Total accumulated savings
- **Purple line** = Number of active savers
- Shows trends over time

### Top Members by Savings
- **Pink-to-Purple bars** = Member savings amounts
- **Tallest bar** = Highest saver
- Helps identify VIP members

### Financial Summary
- **Blue** = Total loaned out
- **Green** = Total paid back
- **Orange** = Interest earned
- **Red** = Outstanding balance
- **Purple** = Total member savings
- **Cyan** = Net position (bottom line)

### Period Comparison
- **Blue bars** = Current month/year
- **Purple bars** = Previous month/year
- Shows if you're growing!

### Repayment Breakdown
- **Green portion** = Principal (original loan amount)
- **Orange portion** = Interest (bank earnings)
- Shows true cost of loans

## Troubleshooting

### Charts not showing?
- Scroll down to Reports section
- Refresh the page (F5)
- Check browser console for errors

### Export not working?
- Try exporting as PNG first
- Check if file downloaded
- Check browser downloads folder

### Data looks wrong?
- Verify member info is correct
- Check loan dates are right
- Ensure payments recorded properly

### Mobile layout broken?
- Rotate phone to landscape
- Use latest browser version
- Clear browser cache
